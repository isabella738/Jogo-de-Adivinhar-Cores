# Jogo-de-Adivinhar-Cores
Joginho de terminal sobre adivinhar as cores e sua ordem

Como funciona:
Existem 6 diferentes cores. O objetivo é acertar uma sequência de 4 dessas cores que podem se repetir ou não
O programa só irá indicar se a cor está contida na sequência, ou se as posições estão certas ou erradas (sem revelá-las diretamente)
São 9 tentativas até acertar
